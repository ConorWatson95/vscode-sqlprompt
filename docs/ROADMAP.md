# Roadmap

Planned next improvements for SQL Prompt.

## Near term

- Column completion after table alias (example: `od.` -> columns from `OrderDetails`).
- Better JOIN assistance with related table suggestions based on foreign keys.
- Optional JOIN clause templates (`ON` conditions prefilled when possible).

## Mid term

- Snippets for `INSERT`, `UPDATE`, and `DELETE`.
- Include stored procedures and views in IntelliSense.
- Hover details for columns and table metadata.
- Go to Definition for table references.

## Future

- Multi-database support in a single workspace session.
- Formatting profiles inspired by SQL Prompt style conventions.

## Notes

Roadmap items are not committed delivery dates. Priority may change based on user feedback and adoption.
